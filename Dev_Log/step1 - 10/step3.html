TelpCamp プロジェクト作成の記録

学習の目的

このステップでは、CRUDの R (Read) のうち、「詳細表示機能」を実装します。 一覧ページから特定のアイテムを選択し、そのアイテムだけの詳細な情報を表示する、Webアプリケーションの基本的なパターンを学びます。

  • URLの一部を可変にする動的なルーティング（URLパラメータ）の使い方を理解する (例: /campgrounds/:id)。
  • リクエストオブジェクト (req.params) からURLパラメータを取得する方法を学ぶ。
  • MongooseのfindById()メソッドを使い、IDを元にデータベースから単一のデータを取得する方法を習得する。
  • サーバーからビューへ単一のオブジェクトを渡し、その内容を表示する方法を理解する。

ステップ3: 詳細表示 (Show) ページの作成

一覧ページでキャンプ場のリンクをクリックした際に、そのキャンプ場だけの詳細情報を表示するページを作成します。

  1. 詳細表示ルートの追加 (index.js):

    /campgrounds/:idという形式のURLにGETリクエストが来た際の処理を追加します。URLに含まれるIDを元にデータベースから特定のキャンプ場を検索し、その情報をビューに渡します。 

    // 詳細(Show)ページへのGETリクエスト
// '/campgrounds/:id' の :id は、URLの一部を変数として受け取るためのプレースホルダーです
app.get('/campgrounds/:id', async (req, res) => {
    // req.paramsオブジェクトからidを取得します
    const { id } = req.params;
    // idを使ってデータベースから特定のキャンプ場を検索します
    const campground = await Campground.findById(id);
    // 見つかったキャンプ場データをビューに渡してレンダリングします
    res.render('campgrounds/show', { campground });
});

    💡 URLパラメータ (:id) と req.params

    ルート定義でコロン(:)を接頭辞として使うと、その部分がプレースホルダーになります。例えば、ユーザーが /campgrounds/60b8... にアクセスすると、Expressは :id の部分に '60b8...' という文字列を当てはめます。この値は req.params オブジェクトに { id: '60b8...' } という形で格納され、コード内から参照できます。

    💡 Campground.findById(id) とは？

    Mongooseが提供する便利なメソッドです。引数に指定したIDを持つドキュメント（今回はキャンプ場のデータ）をデータベースから1件だけ検索してくれます。Campground.findOne({ _id: id }) とほぼ同じ意味ですが、より簡潔に書くことができます。

  2. 詳細表示ビューの作成 (views/campgrounds/show.ejs):

    ルートから渡された単一のcampgroundオブジェクトを元に、詳細ページを生成するEJSファイルを作成します。views/campgroundsフォルダ内にshow.ejsという名前で新規作成します。 

    <!DOCTYPE html>
<html lang="ja">
<head>
    <title><%= campground.name %></title>
</head>
<body>
    <h1><%= campground.name %></h1>
    <h2><%= campground.location %></h2>
    <p>価格: <%= campground.price.toLocaleString() %>円/泊</p>
    <p>詳細: <%= campground.description %></p>

    <footer>
        <a href="/campgrounds/<%= campground._id %>/edit">編集する</a>
        <form action="/campgrounds/<%= campground._id %>?_method=DELETE" method="POST">
            <button>削除</button>
        </form>
        <a href="/campgrounds">キャンプ場一覧へ戻る</a>
    </footer>
</body>
</html>
  3. 動作確認:

    サーバーを再起動（node index.js）し、ブラウザで一覧ページ http://localhost:3000/campgrounds を開きます。一覧からどれか一つのキャンプ場をクリックすると、そのキャンプ場だけの詳細情報が表示されることを確認します。URLが /campgrounds/ユニークなID の形式になっていることも確認しましょう。
