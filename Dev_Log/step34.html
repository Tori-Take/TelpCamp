TelpCamp プロジェクト作成の記録

Step 34: データとユーザーの紐付け（所有権の記録）

アプリケーションのセキュリティをさらに強化するため、どのキャンプ場がどのユーザーによって作成されたかを記録する機能を実装しました。これは、後のステップで「投稿者本人だけが編集・削除できる」といった、より詳細な認可（Authorization）を実装するための重要な基盤となります。

学習の目的

  • Mongooseの参照（ref）を使い、異なるコレクション間のドキュメントを関連付ける方法を学ぶ。
  • キャンプ場作成時に、ログイン中のユーザー情報（req.user）をデータに紐付けて保存する方法を習得する。
  • 詳細表示ページで、関連付けられたユーザー情報をpopulate()を使って取得・表示する方法を実践する。

実装の解説

モデルのスキーマを更新し、ルートのロジックを修正することで実装しました。

1. Campgroundモデルへのauthorフィールド追加

models/campground.jsを修正し、登録者のIDを保存するためのauthorフィールドを追加しました。ref: 'User'と指定することで、このフィールドがUserモデルのドキュメントを参照することを示します。 

// models/campground.js
const campgroundSchema = new Schema({
    // ...
    author: {
        type: Schema.Types.ObjectId,
        ref: 'User'
    },
    // ...
});

2. 新規登録ルートの修正 (routes/campgrounds.js)

キャンプ場を新規作成するPOST /campgroundsルートで、リクエストボディから作成したcampgroundオブジェクトに、ログイン中のユーザーID（req.user._id）をauthorとして設定してから保存するように変更しました。 

// routes/campgrounds.js
router.post('/', isLoggedIn, ..., wrapAsync(async (req, res) => {
    const campground = new Campground(req.body.campground);
    campground.author = req.user._id; // ログインユーザーを登録者として設定
    await campground.save();
    // ...
}));

3. 詳細表示ルートの修正 (routes/campgrounds.js)

詳細ページで登録者名を表示するために、GET /campgrounds/:idルートのfindByIdクエリに.populate('author')を追加しました。これにより、キャンプ場データと一緒に、関連するユーザーの全情報も取得できます。 

// routes/campgrounds.js
router.get('/:id', wrapAsync(async (req, res, next) => {
    const campground = await Campground.findById(id)
        .populate('reviews')
        .populate('author'); // author情報をpopulate
    // ...
}));

この実装により、すべてのキャンプ場データに「所有者」の情報が記録されるようになり、より高度なアクセス制御を行うための準備が整いました。

4. 詳細ページでの登録者表示 (views/campgrounds/show.ejs)

最後に、populateで取得した登録者情報を、詳細ページに表示します。campground.authorにはユーザーオブジェクトが格納されているため、campground.author.usernameのようにしてユーザー名にアクセスできます。 

<!-- views/campgrounds/show.ejs -->
<ul class="list-group list-group-flush">
    <li class="list-group-item text-muted"><%= campground.location %></li>
    <li class="list-group-item">登録者: <%= campground.author.username %></li>
    <li class="list-group-item"><%= campground.price.toLocaleString() %>円/泊</li>
</ul>
