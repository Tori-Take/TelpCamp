Step 42: 詳細ページへの地図表示 (Mapbox Geocoding)

キャンプ場の場所を視覚的に分かりやすくするため、各キャンプ場の詳細ページにMapboxを利用した地図を表示する機能を実装しました。 この機能の核となるのは、ユーザーが入力した地名（例：「東京都庁」）を、地図上に表示可能な緯度・経度の座標に変換する「ジオコーディング」という技術です。

学習の目的

  • Mapbox Geocoding APIを利用して、地名から座標を取得する方法を学ぶ。
  • 非同期処理（async/await）を使い、外部APIからのレスポンスを待って処理を続行する方法を実践する。
  • 取得した座標データを、MongooseのGeoJSON形式（Point）でデータベースに保存する方法を習得する。
  • 詳細ページで、保存された座標データを使ってMapboxの地図とマーカーを表示する方法を学ぶ。
  • APIキーなどの機密情報を環境変数（.env）で管理し、安全性を高める。

実装の解説

実装は、キャンプ場の新規登録・編集時にジオコーディングを行い、詳細ページでその結果を利用して地図を描画するという流れで行いました。

1. Mapbox Geocodingクライアントの準備

MapboxのAPIを利用するためのクライアントをインストールし、アクセストークンを設定しました。 

npm install @mapbox/mapbox-sdk
// index.js or a dedicated config file
const mbxGeocoding = require('@mapbox/mapbox-sdk/services/geocoding');
const mapBoxToken = process.env.MAPBOX_TOKEN;
const geocoder = mbxGeocoding({ accessToken: mapBoxToken });

2. ジオコーディングと座標の保存 (controllers/campgrounds.js)

キャンプ場の新規登録(createCampground)と更新(updateCampground)のコントローラーに、ジオコーディング処理を追加しました。 Mapbox Geocoding APIに地名（req.body.campground.location）を問い合わせ、返ってきた座標をgeometryフィールドに保存します。 

// controllers/campgrounds.js
module.exports.createCampground = async (req, res, next) => {
    const geoData = await geocoder.forwardGeocode({
        query: req.body.campground.location,
        limit: 1
    }).send();
    const campground = new Campground(req.body.campground);
    campground.geometry = geoData.body.features[0].geometry; // 座標を保存
    // ...
    await campground.save();
    // ...
};

3. Campgroundモデルの更新 (models/campground.js)

ジオコーディングの結果（GeoJSON）を保存するため、geometryフィールドをスキーマに追加しました。 

// models/campground.js
const campgroundSchema = new Schema({
    // ...
    geometry: {
        type: {
            type: String,
            enum: ['Point'],
            required: true
        },
        coordinates: {
            type: [Number],
            required: true
        }
    },
    // ...
});

4. 詳細ページでの地図とマーカーの表示 (show.ejs, showPageMap.js)

詳細ページ(show.ejs)にMapboxのライブラリと地図を表示するコンテナを設置しました。 そして、サーバーから渡されたキャンプ場の座標(campground.geometry.coordinates)を使い、地図の中心設定とマーカーの追加を行うJavaScript(showPageMap.js)を読み込みます。 

<!-- show.ejs -->
<div id='map' style='width: 100%; height: 300px;'></div>

<script>
    const mapToken = '<%- process.env.MAPBOX_TOKEN %>';
    const campground = <%- JSON.stringify(campground) %>;
</script>
<script src="/javascripts/showPageMap.js"></script>

// public/javascripts/showPageMap.js
mapboxgl.accessToken = mapToken;
const map = new mapboxgl.Map({
    container: 'map',
    style: 'mapbox://styles/mapbox/streets-v11',
    center: campground.geometry.coordinates, // キャンプ場の座標
    zoom: 10
});

// マーカーを追加
new mapboxgl.Marker()
    .setLngLat(campground.geometry.coordinates)
    .setPopup(
        new mapboxgl.Popup({ offset: 25 })
            .setHTML(`<h3>${campground.title}</h3><p>${campground.location}</p>`)
    )
    .addTo(map);

この実装により、ユーザーはキャンプ場の場所を地図上で直感的に確認できるようになり、利便性が大きく向上しました。また、この座標データは次のステップであるクラスターマップ実装の基礎となります。
